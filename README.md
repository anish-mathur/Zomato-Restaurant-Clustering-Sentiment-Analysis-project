# Zomato-Restaurant-Clustering-Sentiment-Analysis-project
This project focuses on analyzing Zomato restaurant data to understand restaurant characteristics, customer preferences, and engagement patterns in the food industry. Zomato is one of the largest restaurant discovery and food delivery platforms, providing information about restaurants, cuisines, pricing, and customer reviews.
